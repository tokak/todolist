# todolist
Bu projede, kullanıcıların kayıt olup giriş yapabileceği ve sadece kendi görevlerini ekleyip yönetebileceği basit bir To-Do List uygulamasıdır. Backend PHP ve MySQL ile, frontend ise HTML ve Bootstrap 5 kullanılarak geliştirdim.
Projede temel CRUD (Oluştur, Oku, Güncelle, Sil) işlemleri ve kullanıcı oturum yönetimi uygulamalı olarak gerçekleştirildi.
